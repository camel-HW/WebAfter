1. HTML : 웹페이지를 제작하기 위한 도구 (틀)
        2. CSS : HTML 표현한다
        3. JavaScript : HTML 제어한다
            - 다양한 라이브러리를 사용할 수 있다
            - 웹 애플리케이션을 만듬
            - 서버를 구성하고 서버용 프로그램을 만들 수 있음

        4. 자바스크립트 구성
            - 자바스크립트 CORE : 기본문법과 구조, 데이터 타입, 조건문, 반복문, 함수....
            - 자바스크립트 BOM(Browser Object Model) : 브라우저와 관련된 Window, Navigator, Laction, History ...
            - 자바스크립트 DOMDocument Object Model): Element 제어, Style 제어, Event제어, Attribute 제어
