태그의 구성: 태그 또는 텍스트(데이터) 구성된다
태그의 역할 : 데이터 구분자 역할, 기능적 역할
태그는 속성을 가질 수 있다.
글꼴, 문단및 목록, table : CSS 대체 가능
이미지, 영상, 오디오 : 상대 경로
페이지 이동 : a 태그 사용
